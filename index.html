¿Por qué Python?

Python es uno de los lenguajes más usados del mundo, ¿pero por qué?

La principal razón es por ser un lenguaje multiplataforma e interpretado. ¿Cuáles son sus ventajas? Aquí una tabla con las diferencias entre lenguaje compilado e interpretado

Tipos Ventajas Desventajas
Compilado Puede separarse del entorno de desarrollo, ejecutarse de forma independiente en una plataforma específica y tiene una mayor eficiencia operativa. No se puede trasplantar; es necesario trasplantar, recopilación del código fuente.
Interpretado La multiplataforma es más fácil, a expensas de la eficiencia de ejecución del programa. Baja eficiencia, no se puede ejecutar de forma independiente del intérprete

Tipos de Datos

En Python, existen diferentes tipos de datos:

  • Datos básicos:
    • Los booleanos
    • Los numéricos (enteros, punto flotante y complejos)
    • Las cadenas de caracteres
  • Otros tipos de datos:
    • Secuencias: Los tipos list, tuple y range
    • Mapas: El tipo dict
    • Conjuntos: El tipo set
    • Iteradores
    • Clases
    • Instancias
    • Excepciones

Datos Básicos

Los booleanos

Al igual que en otros lenguajes de programación, en Python existe el tipo bool o booleano. Es un tipo de dato que permite almacenar dos valores True o False.

Los numéricos

Python distingue entre enteros, números de punto flotante y números complejos.

  • Int: Número entero con precisión fija.
  • Long: Número entero en caso de overflow.
  • Float: Coma flotante de doble precisión.
  • Complex: Parte real y parte imaginaria j.

Las cadenas de caractéres

Se almacenas caractéres en forma de cadena, pudiendo, por ejemplo, de una palabra ser utilizada cada letra individualmente.

Datos Básicos

Secuencias

Existen varios tipos:

  1. List: Para definir una, se debe escribir es entre corchetes, separando sus elementos con comas cada uno.
  2. Tuple: La clase tuple en Python es un tipo contenedor, compuesto, que en un principio se pensó para almacenar grupos de elementos heterogéneos, aunque también puede contener elementos homogéneos.
  3. Range: Es un tipo de dato que representa una secuencia de números inmutable.

Mapas

La clase dict de Python es un tipo mapa que asocia claves a valores. A diferencia de los tipos secuenciales (list, tuple, range o str), que son indexados por un índice numérico, los diccionarios son indexados por claves.

Funciones principales

Print()

En los programas, para que python nos muestre texto o variables hay que utilizar la función print(). La función print() permite mostrar texto en pantalla. El texto a mostrar se escribe como argumento de la función:

Ejemplo: print("Hola Mundo") --> Output: Hola Mundo

If()

La estructura de control if ... permite que un programa ejecute unas instrucciones cuando se cumplan una condición. En inglés "if" significa "si" (condición).

La sintaxis de la construcción if es la siguiente:

  if condición:

    aquí van las órdenes que se ejecutan si la condición es cierta

    y que pueden ocupar varias líneas

Input()

La función input() permite obtener texto escrito por teclado. Al llegar a la función, el programa se detiene esperando que se escriba algo y se pulse la tecla Intro, como muestra el siguiente ejemplo:

  variable = input("Introduce tu nombre: ")
